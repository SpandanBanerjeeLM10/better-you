# BETTER-YOU
A full-stack Self Help website using HTML, CSS, JS, Springboot, SpringSecurity, and MySQL.

# Front-End
  Developed using HTML, CSS, and JS.
  Dynamic & Interactive web elements made using JavaScript.
  API Connections are made using JavaScript to send and receive data using REST APIs.
  
# Back-End
  Using Springboot framework.
  
# Spring-Security
  Added Spring-Security for a secure login functionality with DB connectivity.
